თქვენი სიტყვებით ახსენით რა არის container და empty ელემენტები

container tag თეგია რომელსაც აქვს გამხსნელი თეგიც და დამხურავი თეგიც.
empty tag თეგია რომელსაც მხოლოდ გამხსნელი თეგი აქვს.


თქვენი სიტყვებით ახსენით რა არის კომენტარები და ახსენეთ როდის ვიყენებთ მათ. 
მოიძიეთ 5 განსხვავებულ პროგრამირების ენაში ან ტექნოლოგიაში როგორ იწერება კომენტარები, მაგალითად: python, html, css, js, c, c++, java და ა.შ.


კომენტარი გვეხმარება კოდის წერის დროს რომ გამოვყოთ კოდები სექციებათ. 
კომენტარი კარგია თვითონ პროგრამერისთვის და თვითონ იმისთვისაც ვინც ეს კოდი უნდა წაიკითხოს და შეამოწმოს.
მას ვიყენებთ კოდის გამოსაყოფად ასევე კომენტარით შეგვიძლია კომპიუტერს ჩვენი დაწერილი კოდი გავუქროთ მაგრამ ჩვენ მას კიდევ დავინახავთ.
ასევე გვეხმარება კოდის შესწორებისთვისაც როდესაც არ გინდა ქვედა კოდის ნაწილმა ხელი არ შეგიშალოს.

python-ში კომენტარი იწერება ძიეზით(#).
html-ში არის გამოყოფელი სპეციალური თეგი:<!--აქ ჩაიწერება ტექსტი-->.
css-ში კომენტარი ჩაიწერება ასე:  /*აქ ჩაიწერება ტექსტი */
javascript-ში  //ტექსტი აქ 
c-ში არის 2 სახის კომენტარი თუ 1 ხაზზეა კომენტარი მაშინ //აქ ტექსტი. თუ ბევრი რამდენიმე ხაზია მაშინ /*ტექსტი აქ*/
c++-შიც იგივე ნაირად როგორც c ში ანუ // და /* */
java-ში იგივე ნაირად როგორც c-ში და c++
