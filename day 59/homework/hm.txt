4. ახსენით js-ის კოდის დაწერა რამდენ განსხვავებულ ადგილას შეიძლება, მაგ. ატრიბუტში, script თეგში და ა.შ. ჩამოწერეთ თითოეული ასეთი მიდგომის სახელი

js-ის კოდის ჩაწერა შეიძლება ჩაიწეროს სამი მიდგომით ესენია:inline,external,internal.

Inline JavaScript – კოდი ჩაწერილია HTML ელემენტის ატრიბუტში.

Internal JavaScript – კოდი ჩაწერილია <script> ტეგში იმავე HTML დოკუმენტში.

External JavaScript – კოდი ინახება ცალკე ფაილში და იმპორტირდება HTML დოკუმენტში src ატრიბუტით.
