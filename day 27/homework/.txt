2.  გააკეთეთ ნახაზი css-ის კოდის წერის ტექნიკების (inline, internal, external) შესახებ

(ნახაზი არის homework folder-ში)


3. თქვენი სიტყვებით ახსენით რა არის დეკლარაცია, კუთვნილება, მნიშვნელობა, css rule, selector, css inheritance, descendant selector


დეკლარაცია არის სტილის მითითების პროცესი, როდესაც ჩვენ ვაკავშირებთ  კუთვნილებას და მის მნიშვნელობას რაიმე ელემენტისთვის.

კუთვნილება ანუ იგივე property არის ის მახასიათებელი რომელიც განსაზღვრავს სტილის ტიპს მაგ:color,background-color,font size და ასე შემდეგ.

მნიშვნელობა არის ის მონაცემი, რომელიც მიენიჭება კონკრეტულ კუთვნილებას მაგ: font-size:small; , color:green; და ასე შემდეგ.

CSS rule არის ინსტრუქცია რომელიც ეუბნება ბრაუზერს როგორ უნდა გამოიყურებოდეს ელემენტი.

selector-ი იღებს იმ ელემენტს რომლის გასტილვაც შენ გინდა.

CSS inheritance არის მექანიზმი, რომლის საშუალებითაც შვილ ელემენტებს მოჰყვებათ სტილი მათი მშობელი ელემენტებისგან.

Descendant selector გამოიყენება შვილი ელემენტების სტილის განსასაზღვრად, რომლებიც სხვა ელემენტების შიგნით არის.
