2. თქვენი სიტყვებით ახსენით რა განსხვავებაა id და class ატრიბუტებს შორის, ასევე ახსენით როდის რომლის გამოყენება სჯობს


id და class ატრიბუტებს შორის ის განსხვავებაა რომ selector-ის დაწერის დროს id-ს ვუთითებთ როგორც "#" ხოლო class-ის დროს ".".
id-ის გამოყენება სჯობს მაშინ როდესაც ჩვენ გვინდა ერთი ელემენტი გავსტილოთ რადგან ერთიადიგივე id რამდენიმე ელემენტს ვერ მივანიჭებს ხოლო class-ს ჯობია გამოვიყენოთ როდესაც 
რამდენიმე ელემენტის გასტილვა გვინდა რადგან class-ი შეგვიძლია ბევრ ელემენტს გავუწეროთ.





5. თქვენი სიტყვებით ახსენით რა არის css specificity და ჩამოწერეთ პრიორიტეტების სია


css specificity არის ერთგვარი ალგორითმი რომელიც გამოიყენება ბრაუზერებში რადგან ბრაუზერი მიხვდეს რომელი გამოიყენოს. ანუ ელემენტზე გაწერილი კოდში რომ იყოს: #p1 და .p2 ორივე რომ იყოს
გასტილული ის ამოირჩევს #p1 ანუ id-ს რადგან id-ი class-ზე პრიორიტეტულია.
